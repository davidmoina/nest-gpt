## Description

Is the backend for the react gpt app

## Installation

1. Clone the repository

2. Install dependencies

```bash
$ npm install
```

3. Copy .env file and put your variables

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
